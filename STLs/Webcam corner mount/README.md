# Trident corner mount web cam  

This is my web cam mount design I created for my Trident build. It's made for the Arducam, or anything of similar size-
https://www.amazon.com/dp/B0829HZ3Q7?ref_=ppx_hzsearch_conn_dt_b_fed_asin_title_7

There is a channel on the outside of the perimeter for running chamber light wiring or thermistors. The larger diameter hole is for running wiring to a top mounted electronics bay. The Cam holder is attached to the cam corner mount via 2x m3 heatserts and appropriate fasteners.
